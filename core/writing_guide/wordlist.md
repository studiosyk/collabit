# wordlist

このファイルには、Collabit の記事執筆で使う単語、表記、表記ゆれルール、
推奨語、禁止語、短い言い換えを管理します。

このファイルに入れるもの:
- 固有名詞の正式表記
- 表記ゆれを防ぐためのルール
- 推奨語 / 禁止語
- 記事内で統一したい短い言い回し
- 単語レベルの言い換え

このファイルに入れないもの:
- 文章全体のトーン
- です・ます調などの文体方針
- 比喩を使うかどうか
- 断定の強さの考え方

## Purpose

- 推奨する単語や表記をそろえる
- 避けたい表現を明確にする
- 表記ゆれを防ぐ
- 単語レベルの言い換え候補を共有する

## Term Rules

使う表記と避ける表記を、対で管理します。

| 種別 | 正式名称 | 基本表記 | 避ける表記 | 補足 |
| --- | --- | --- | --- | --- |
| 名詞 |  | `査定額` | `査定金額` | `査定額` に統一する |
| ブランド名 | `HowMa` | `HowMa` |  | 記事内の基本表記 |
| 会社名 | `株式会社コラビット` | `コラビット` | `コラビット社` | 初出や正式名称が必要な文脈では `株式会社コラビット`、通常文中では `コラビット` を使う |

## Style Choices

単語そのものではなく、数字、記号、英数字などの書式ルールを書きます。

| 項目 | 基本ルール | 補足 |
| --- | --- | --- |
| 数字表記 | 必要に応じて追記 | 例: 半角数字を使う |
| 記号表記 | 必要に応じて追記 | 例: カッコ、スラッシュ、コロンの使い方 |
| 英数字表記 | 必要に応じて追記 | 例: 英単語の大文字小文字ルール |
