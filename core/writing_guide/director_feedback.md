# director feedback

このファイルは、複数記事にまたがって有効なディレクター指摘を蓄積するための記録です。

## Purpose

- 記事単位ではなく、案件全体で繰り返し参照したい指摘を残す
- 文体、構成、訴求、CTA、表現ルールの改善点を共有する
- 同じ指摘を何度も繰り返さないようにする

## Usage

- その記事だけに関する指摘は、各記事フォルダの `brief.md` または `log.md` に書く
- 複数記事に共通する指摘だけを、このファイルに転記する
- 新しい指摘を追加する時は、日付と文脈が分かるように残す
