# collabit core

Collabit 本体の継続運用領域です。

## Structure

- `writing/`
  - 記事、構成案、調査メモ、公開原稿を扱います。
- `writing_guide/`
  - 執筆レギュレーション、トンマナ、構成ルール、単語管理を扱います。
- `design/`
  - 提案資料、ビジュアル、図版などの制作物を扱います。
- `design_guide/`
  - `design_system.md` を中心に、ブランド全体と記事アセットのデザイン基準を扱います。
- `dev_in/`
  - 自分用の補助ツールや内部利用前提の仕組みを扱います。
