# HowMa執筆辞書

自分用の執筆補助メモ。HowMa記事を書くときに、用語の説明不足や業界語の置きっぱなしを防ぐために使う。

## 使い方

- 初見読者が止まりそうな語を先に確認する
- 本文でそのまま使うか、言い換えるか、補足を添えるかを判断する
- 記事ごとに必要なら項目を追記する

## 補足が必要な用語

### 査定まわり

- `AI査定`
  - そのまま使うときの注意:
    - 訪問査定と同じものだと思われることがある
  - 補足のしかた:
    - `相場の目安をつかむための査定`
    - `まず価格感を知るための査定`

- `一括査定`
  - そのまま使うときの注意:
    - サービス内容が伝わらないことがある
  - 補足のしかた:
    - `複数の不動産会社にまとめて査定依頼できるサービス`

- `机上査定`
  - そのまま使うときの注意:
    - 言葉だけでは意味が伝わりにくい
  - 補足のしかた:
    - `現地訪問なしで、物件情報や周辺相場から出す査定`

- `訪問査定`
  - そのまま使うときの注意:
    - 机上査定との違いを補わないと分かりにくい
  - 補足のしかた:
    - `実際に現地を見たうえで行う査定`

### 会社・契約まわり

- `地場`
  - そのまま使うときの注意:
    - 一般読者には伝わりにくい
  - 言い換え候補:
    - `地域密着型の不動産会社`
    - `地域に根ざした不動産会社`

- `専任`
  - そのまま使うときの注意:
    - `専任媒介契約` を知らない読者には意味が飛ぶ
  - 補足のしかた:
    - `1社に優先して売却を任せる契約形態`

- `一般媒介`
  - そのまま使うときの注意:
    - 専任との違いが分からないと意味が伝わらない
  - 補足のしかた:
    - `複数の会社に同時に売却を依頼できる契約形態`

### 売却活動まわり

- `囲い込み`
  - そのまま使うときの注意:
    - 初見だと意味が分からない
  - 補足のしかた:
    - `他社経由の買主紹介を積極的に受けないことで、売却機会が狭まること`

- `レインズ`
  - そのまま使うときの注意:
    - 一般読者は知らない言葉として扱う
  - 補足のしかた:
    - `不動産会社同士が物件情報を共有する仕組み`

- `チャレンジ価格`
  - そのまま使うときの注意:
    - 業界寄りで、意味がぼんやりしやすい
  - 言い換え候補:
    - `まずは高めに売り出す価格設定`

- `プランB`
  - そのまま使うときの注意:
    - 中身が見えない
  - 言い換え候補:
    - `売れなかったときの次の打ち手`
    - `次の対策`

- `販売戦略`
  - そのまま使うときの注意:
    - 抽象語になりやすい
  - 具体化の観点:
    - 価格設定
    - 広告の出し方
    - いつ見直すか
    - 売れないときの対応

## 言い換えたい表現

- `営業マン`
  - 推奨:
    - `担当者`
    - `営業担当`

- `味方かどうか`
  - 推奨:
    - `売主の事情を踏まえて提案してくれるか`
    - `信頼して任せられるか`

- `ダメ営業`
  - 推奨:
    - `避けたい対応をする担当者`
    - `任せにくい担当者`

- `怪しい会社`
  - 推奨:
    - `慎重に見極めたい会社`
    - `説明や対応に不安が残る会社`

- `ガチャ`
  - 推奨:
    - `担当者による差`
    - `担当者との相性`
