今回は、賃貸物件から退去する際に発生する原状回復工事の費用負担について取り上げます。

昨今では、貸主と借主の間で負担割合を巡るトラブルが増加傾向にあり社会問題となっています。

トラブルを事前に防ぐためには、原状回復の概念や工事費用の基本的な負担区分について理解しておくことが重要です。

なお、本コラムは東京都都市整備局が発表している「賃貸住宅トラブル防止」の内容に沿って紹介します。

## 原状回復工事の基本概念

まずは原状回復工事の基本的な考え方から見てみましょう。

原状回復とは、「賃借人の居住、使用により発生した建物価値の減少のうち、賃借人の故意・過失、善管注意義務違反、その他通常の使用を超えるような使用による損耗・毀損を復旧すること」をいいます。

これをもっとわかりやすく解釈すると、借主は通常の使用によって発生する損耗や毀損は負担しなくてもいいということになります。

基本的に賃貸借契約では、貸主と借主の間で上記のような内容に縛られず独自に費用負担を定めることは可能ですが、あまりにも常識から逸脱した貸主有利の場合は、無効になるケースもあります。

借主の費用負担の目安としては、契約時に貸主に預けた敷金（家賃の1～2か月分）の金額以内で収まるのが通常です。

ちなみに東京都では独自に「賃貸住宅防止条例」という条例が定められています。

これは契約時に原状回復工事の費用負担について、不動産会社から書面で説明する義務があることを定めています。

負担割合が借主に偏っていないかチェックするとともに、疑問に思うことがあれば仲介した不動産会社に問い合わせてみましょう。

なお、東京都都市整備局でも原状回復工事費用のトラブル相談窓口が設置されているので、困った時などは利用してみるのも良いかもしれません。

## これって貸主負担？借主負担？

次にどのような工事が貸主負担、借主負担となるのか、ここでは事例をいくつか紹介しながら確認していきたいと思います。

なお、下記はあくまで基本的な負担区分を示すものであり、契約書で定めた内容が優先されますのでご注意ください。

### ＜壁・クロスの補修＞

日照などの自然現象による壁・クロスの変色 → 貸主負担

画鋲やピン等の穴（下地ボードの張り替えが不要な程度） → 貸主負担

壁に張ったポスターや絵画の跡 → 貸主負担

冷蔵庫や電子レンジの後部壁面の黒ずみ（電気やけ） → 貸主負担

喫煙によるタバコのヤニなどによる重度な変色 → 借主負担

### ＜床・フローリング・畳の補修＞

日照などによる変色、家具の設置による床のへこみ、設置跡 → 貸主負担

キャスター付き椅子などによるへこみ、キズ、過失による床のシミや汚れ → 借主負担

### ＜水まわりの補修＞

清掃や手入れを怠って発生した風呂、トイレ、洗面台の水垢やカビ → 借主負担

ガスコンロ置き場や換気扇の油汚れ、スス → 借主負担

### ＜その他の補修＞

鍵の交換費用（借主による毀損や紛失などが無い場合） → 貸主負担

部屋全体のハウスクリーニング（借主が通常の清掃を実施している場合） → 貸主負担

飼育しているペットの引掻きキズや汚れ → 借主負担

結露を放置したことによって発生したカビやシミ → 借主負担

## トラブルが起きないために

いかがでしたでしょうか？

きっと皆さんが想像していた以上に貸主負担が多かったのではないでしょうか？

特に「鍵の交換費用」や「画鋲穴や電気やけ」、「ハウスクリーニング」は借主負担が当然と思っている方が多いようです。

実はこれらは貸主負担が原則なのです。

後になってトラブルが発生しないためにも、契約時に不動産会社の言うことをただ鵜呑みにするのではなく、きちんと内容を把握することが大事です。

最低でも、賃貸借契約書に原状回復工事の負担区分が明確に記載されているかの確認、入居前に部屋の写真を撮っておいて壁や床などのキズが入居前からあったことを記録する等の対策を講じておくほうが良いでしょう。
