# collabit design

Collabit のデザイン作業領域です。

## Rules

- ラフ、要件メモ、参考画像は `inbox/` から着手する
- 制作中のデータは `in_progress/` に置く
- 確認待ちや修正待ちは `review/` に移す
- 納品版や確定版は `final/` に置く
- 役目を終えた案や旧版は `archive/` に移す

## Structure

- `inbox/`
  - 未整理の依頼、参考画像、ラフメモ
- `in_progress/`
  - 制作中のデザインデータ
- `review/`
  - 確認待ち、赤入れ待ち、修正版
- `final/`
  - 確定版、納品版
- `archive/`
  - ボツ案、旧版
- `assets/`
  - ロゴ、図版、素材、書き出し画像
- `_templates/`
  - デザイン依頼や制作メモのひな形
